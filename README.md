# Static Angular Ionic

![](https://img.shields.io/badge/angular-12-red) ![](https://img.shields.io/badge/ionic-6-lightskyblue)

Exploring how to create static websites with Angular + Ionic. See demo (https://lightzane.github.io/static-angular-ionic)

This project was generated via `@angular/cli` first then added [Ionic](https://ionicframework.com/).

## Getting Started

```
ng new my-app
ng add @ionic/angular
```

## Then Update the Following:

- [index.html](#update-html)
- [angular.json](#update-angular-configuration)
- [variables.scss](#replace-theme-variables)
- [app-routing.module.ts](#update-app-routing-module)
- [app.component.html](#update-app-component-html)

```diff
{

+   "input": "src/theme/variables.scss"
}
```
